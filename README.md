
# Markov Text Generator

This project implements a simple Markov Chain based text generator using Python.

## Description
The program reads text data from a file and generates new sentences based on the probability of word sequences.

## Files
- markov_text_generator.py : Python script for generating text
- data.txt : Training dataset used for the Markov model
- output.png : Example output of generated text

## How to Run
1. Open terminal
2. Run the following command:

python markov_text_generator.py

## Output
The program generates new sentences based on the patterns learned from the dataset.
